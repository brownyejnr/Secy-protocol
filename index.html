Help & FAQ
What is OP_NET?
OP_NET is a smart-contract layer running directly on Bitcoin L1, enabling trustless DeFi without bridges or sidechains.
How do I connect my wallet?
Click "Connect OP_WALLET" in the top-right corner. Secy Protocol uses the official OP_WALLET browser extension for all transaction signing.
What is Floor Price?
The lowest active listing price in a collection. It reflects the minimum you'd pay to instantly acquire any item from that collection.
What is APY?
Annual Percentage Yield — the yearly interest rate you earn on supplied assets, compounding linearly per block.
What is LTV?
Loan-to-Value ratio. If your LTV exceeds 75%, your position may be liquidated to protect the pool. Keep it below 60% for safety.
How do Flash Loans work?
Borrow any amount within a single block with zero collateral — as long as you return it plus a 0.09% fee in the same transaction.

⚡ FLASH LOANS

Flash loans allow you to borrow any amount of liquidity from the SecyLend pool within a single Bitcoin block, with zero collateral required.

HOW IT WORKS
1. Borrow up to pool liquidity in one call
2. Execute your arbitrage / liquidation logic
3. Return funds + 0.09% fee in the same tx
4. If repayment fails → tx reverts completely

⚠ Flash loans are an advanced feature designed for developers and arbitrageurs. Improper use will result in full transaction reversion and loss of gas fees.
